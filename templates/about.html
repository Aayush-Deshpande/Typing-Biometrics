How it Works

Continuous Behavioral Biometrics via Keystroke Dynamics

1

Real-Time Capture

The browser monitors raw keyup/keydown event sequences down to millisecond-level precision, tracking key dwell times and inter-key flights.

2

Siamese LSTM Embeddings

An LSTM network (trained on the CMU Keystroke Benchmark and fine-tuned) processes raw sequences to produce distinct 16-dimensional typing embeddings.

3

Profile Clustering

DBSCAN identifies stable patterns in user enrollment samples, clustering them into profiles that represent the user's authentic typing signature.

4

Behavioral Verification

During login, typing patterns are verified using cosine similarity against the profile clusters, rejecting samples that fall below the calibrated threshold.

5

Continuous Auth

Active session validation uses an Exponential Moving Average (EMA) to maintain a rolling Authentication Confidence Score. If hijacked, the system lockouts the session.

Pipeline Architecture

End-to-end data flow from real-time capture to continuous authentication decision.
